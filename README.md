📦 Clash.MI 使用教程（最像小火箭）

特点：界面简洁、无广告、高度兼容 Shadowrocket 配置，iOS 用户首选！
操作步骤：

1. 打开 Clash.MI
2. 点击底部 「Profiles」（我的配置）
3. 点右上角 「＋」→ 选择「Add URL」
4. 粘贴你的订阅链接（提前复制好）
5. 填写备注（如“我的订阅”），点击 ✔ 保存

7. 回到首页，开启右上角 开关，状态显示 “已连接” 即成功！



📦 V2Box 使用教程（新手最友好）

特点：操作极简、支持多协议（SS / SSR / VMess / VLESS / Trojan 等），跨平台（iOS + Android）
操作步骤：

1. 打开 V2Box
2. 进入 「配置」 页面
3. 点击右上角 「＋」
4. 选择 「添加订阅」
5. 粘贴订阅链接，填写名称（如“我的订阅”）
6. 点击 「新增订阅源」 → 若显示节点列表，即导入成功
7. 返回首页，选择任意节点，滑动底部开关开启连接
8. 出现绿色提示 “已连接” 即表示成功！

⚠️ 首次使用会弹出系统 VPN 权限请求，请点击「允许」
